
# Arcade Game

## An arcade game built with Godot Engine

### Changelog

[v0.2] - Defenders and losing
- The ball is now kicked off instead of randomly placed on the field
- Added defenders who chase the ball
- Added tackle mechanics and losing condition

[v0.1] - Initial development
- Added player movement and ball pickup mechanics
- Added touchdown and scoring mechanics

### Features
- [X] Random kickoff from endzone to place on field
- [X] Player taps on ball to grab it and taps to run
- [X] Player tries to dodge defenders who run at him and try to tackle him
- [X] If you get into the endzone, you score a touchdown it resets
- [X] Keep kicking off and returning until you are tackled X times
- [X] Keep track of score and tackles
