
# Arcade Game

## An arcade game built with Godot Engine

### Features
- [] Random kickoff from endzone to place on field
- [] Player taps on ball to grab it and taps to run
- [] Player tries to dodge defenders who run at him and try to tackle him
- [] If you get into the endzone, you score a touchdown it resets
- [] Keep kicking off and returning until you are tackled X times
- [] Keep track of score and tackles
